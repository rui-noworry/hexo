### 移动端web的基础知识

		（mobile）font-size == 12px （pc）约等于

### viewport的content各个值和属性

		width=device-width, // 页面宽度=设备宽度
		initial-scale = 1.0，// 初始缩放
		maximum-scale = 1.0, // 最大缩放
		user-scalable  = no // 用户是否能缩放

### rem与px转换技巧

>1rem = 设计图宽度/20或10 px （根据情况而定）
